# 🚀 빠른 시작 가이드 (Quick Start)

## Forklift Management System V6 - GitHub Pages 배포

### ⚡ 5분 만에 배포하기

#### 1️⃣ 파일 다운로드
- `forklift-system-v6-github-pages.zip` 다운로드 및 압축 해제

#### 2️⃣ GitHub 저장소 생성
1. [GitHub](https://github.com) 로그인
2. 우측 상단 **+** → **New repository** 클릭
3. Repository name: `forklift-management` (원하는 이름)
4. **Public** 선택
5. **Create repository** 클릭

#### 3️⃣ 파일 업로드
1. 생성된 저장소에서 **uploading an existing file** 클릭
2. 압축 해제한 파일들을 드래그 앤 드롭:
   - ✅ `index.html` ⭐ (필수!)
   - ✅ `README.md`
   - ✅ `DEPLOYMENT_GUIDE.md`
   - ✅ `DEPLOYMENT_CHECKLIST.md`
3. 하단에 "Initial commit" 입력
4. **Commit changes** 클릭

#### 4️⃣ GitHub Pages 활성화
1. 저장소 상단 **Settings** 탭 클릭
2. 좌측 메뉴에서 **Pages** 클릭
3. **Source** 섹션:
   - Branch: **main** 선택
   - Folder: **/ (root)** 선택
4. **Save** 클릭

#### 5️⃣ 완료! 🎉
- 1-2분 후 다음 URL로 접속:
  ```
  https://YOUR_USERNAME.github.io/forklift-management/
  ```
- 예시: `https://johndoe.github.io/forklift-management/`

## 🛠️ 문제 해결

### 페이지가 404 오류?
→ 파일 이름이 정확히 `index.html`인지 확인

### 이미지가 안 보여요?
→ 이미 Base64로 임베드되어 있어 정상입니다!

### 기능이 작동하지 않아요?
→ 브라우저 콘솔(F12)에서 오류 확인

### 배포가 진행되지 않아요?
→ Settings → Pages에서 배포 상태 확인

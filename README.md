# Forklift Management System V6

## 🚀 Samsung SDS Cello - Forklift Management System

완전한 기능을 갖춘 포크리프트 관리 시스템입니다.

### ✨ 주요 기능

- **실시간 대시보드**: 포크리프트 상태 모니터링
- **체크리스트 관리**: 일일 점검 및 안전 검사
- **정비 기록**: 정비 이력 추적 및 관리
- **보고서 생성**: Excel 및 PDF 보고서 자동 생성
- **데이터 분석**: 운영 효율성 분석 및 시각화

### 📋 사용 방법

1. **로그인**
   - 사용자명: `admin`
   - 비밀번호: `admin123`

2. **대시보드**
   - 전체 포크리프트 상태 확인
   - 실시간 통계 조회

3. **체크리스트**
   - 일일 점검 항목 작성
   - 안전 점검 기록

4. **정비 관리**
   - 정비 스케줄 관리
   - 정비 이력 추적

5. **보고서**
   - Excel/PDF 보고서 다운로드
   - 데이터 분석 및 시각화

### 🛠️ 기술 스택

- **Frontend**: React 18
- **UI**: Custom CSS with animations
- **Export**: SheetJS (Excel), jsPDF (PDF)
- **Storage**: Browser LocalStorage

### 📦 배포

이 시스템은 단일 HTML 파일로 구성되어 있어 별도의 빌드 과정이 필요 없습니다.

#### GitHub Pages 배포 방법:

1. 이 저장소를 포크하거나 클론합니다
2. GitHub 저장소 Settings → Pages로 이동
3. Source를 "Deploy from a branch" 선택
4. Branch를 "main" (또는 master) 선택
5. 저장 후 배포 URL 확인
